# 🎨 AI Thumbnail Pro — Auto Social Media Graphics

**AI generates beautiful thumbnails, covers, and social cards. 8 presets, 8 color palettes, zero human input.**

## What Is This?

A Python engine running on a $35 Raspberry Pi that automatically generates social media graphics using Pillow. Feed it a title, pick a preset, get a professional image.

Supports: YouTube thumbnails, blog featured images, Twitter/X cards, LinkedIn posts, Instagram squares, Stories — all with gradient backgrounds, auto text layout, and decorative elements.

## Quick Start

```bash
# Single thumbnail
python3 engine.py "How AI Changed My Life" --preset youtube

# Batch — all presets at once
python3 engine.py "Product Launch Day" --batch

# With subtitle
python3 engine.py "AI Automation" --preset blog --subtitle "Built on $35 Raspberry Pi"
```
